# Linear Regression

A minimal Python project scaffold for experimenting with linear regression models (scikit-learn).

## What’s included

- `src/linear_regression/` — package skeleton
- `notebooks/Linear_Regression.ipynb` — blank Jupyter notebook to start experiments
- `data/`, `models/`, `tests/`, `docs/` — helpful folders for project organization
- `requirements.txt` — dependencies

## Quick start (Windows PowerShell)

1. Create a virtual environment:

```powershell
python -m venv .venv
```

2. Activate the environment (PowerShell):

```powershell
.\.venv\Scripts\Activate.ps1
```

3. Install dependencies:

```powershell
pip install -r requirements.txt
```

4. Run Jupyter Notebook:

```powershell
jupyter notebook
```

Open `notebooks/Linear_Regression.ipynb` to begin.
